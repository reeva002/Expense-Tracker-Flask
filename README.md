# AI-Powered Expense Tracker

An AI-powered expense tracking web application built using Flask and Google Gemini AI.

## Features

- Add and manage expenses
- AI automatically predicts expense category
- AI-generated financial insights
- Budget monitoring
- Responsive UI
- Secure API key management using .env
- Deployed on Render

## Tech Stack

- Python
- Flask
- HTML
- CSS
- Google Gemini API
- Gunicorn
- Render
